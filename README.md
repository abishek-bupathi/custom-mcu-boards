### Features
- Bare Minimum General Purpose (BMGP) board based on STM32F103C8T Microcontroller
- 16 Mhz Clock
- Operating at 3.3 V
- Peripherals: 1xI2C, 1xSPI, 1xUART
- Connectivity: USB 2.0 Type C
